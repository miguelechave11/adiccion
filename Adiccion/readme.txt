Enunciado:
	Escribir  los pasos para mostrar la suma de dos numeros que ingreso el usuario.

Analisis:
	Es un problema que necesita datos para procesar, una vez procesado devuelve un resultado.Se necesitan como minimo dos numeros, se suman y se muestra el resultado.

Pasos a seguir:
	>Pedir un numero y almacenarlo en una variable
	>Pedir otro numero y almacenarlo en otra variable
	>Sumar ambos numeros 
	>Mostrar el resultado en pantalla
